0. 코딩 환경 준비하기

1. Code editor 준비

- 시기에 따라서 항상 새로운 tool이 개발되므로 항상 그 당시의 최고의 Editor를 검색해서 사용할 수 있다.
- 구글에서 best web editor 등으로 검색
- 2023년 기준으로 : ATOM, Vim, VS Code 등등


=> 이전에 다운받아놓은 것이 있는 VS Code로 결정

I. Visual Studio Code

1. 다운로드 사이트(https://code.visualstudio.com)

- 운영체제에 맞는 것으로 다운 및 설치
- 언어팩 설치해서 한국판으로 설치해도 무방

2. 작업 공간(폴더) 만들기

1) 윈도우 파일탐색기와 같다고 생각하고 적당한 곳에 작업 폴더를 만든다
2 )오류 가능성을 줄이기 위해서 가능하면 상위디렉토리에 만들고, 영문 이름으로 한다
3) New file 만들기 => 확장자는 html로 한다

3. Live server 설치

1) 반드시 설치할 필요는 없지만, 작업 컴을 서버로 하여, 실시간으로 결과 웹페이지를 확인가능
2) 좌측 그림 메뉴에서 Extensions(확장) => Live server 검색 및 설치 => 우측 하단 Go live 실행

4. 사용 팁 및 장점

1) 항상 저장하는 습관을 들여라(Ctrl + S)
2) 태그 일부만 쳐도 추천 및 완성을 해준다
3) html5은 html 헤드 양식을 만들어준다
4) head+body => head 태그 body 태그를 자동으로 만들어준다
5) 태그*n => 태그 n개를 만들어준다
6) tab 키는 한번에 4칸씩 오른쪽으로 이동
7) Shift + Tab은 왼쪽으로 4칸씩 이동
8) 주석은 언어에 따라 다르지만, Ctrl + / 는 알아서 주석표시를 해준다. 또 누르면 사라진다 ( 한 줄 주석 단축키 )
9) 블록 주석 단축키 : alt + shift + a
10) 단어 찾기 단축기 : Ctrl + F => 단어 수정도 가능, 하나 또는 전체
11) 파일 찾기 단축기 : Ctrl + P
12) 전체찾기 : 왼쪽 돋보기 메뉴

5. 설정

1) File => Preferences => Settings 에서
2) zoom => 작업 공간 크기 조절
3) word wrap on => 자동 줄 바꿈
4) 메뉴또는 명령어 검색 : View => Command Palette

6. 실행

1) Run and Debug => F5
